# 📁 Input Materials (사전 참고 자료 폴더)

사용자(Human)가 작성할 리포트의 **원천 데이터(Source Data)나 참고할 예시 형식(PDF, CSV, 텍스트 등)**을 이 폴더에 던져두시면 됩니다.

**WizWriting Supporter의 AI 에이전트(탐색가, 작성가 등)는 이 폴더를 최우선으로 스캔합니다.** 
인터넷 검색보다 먼저 이 폴더 내의 파일들을 읽고 분석하여 문서를 작성합니다.

- 팁 1: 원하는 리포트의 "예시 양식" 파일을 2개 이상 넣어두면 에이전트가 그 형식을 완벽하게 흉내 냅니다.
- 팁 2: 로우 데이터(raw data)를 넣어두면 탐색가가 알아서 데이터를 추출하여 표나 차트로 가공합니다.
